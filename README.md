# API DocsThis is a repo for managing machine readable artifacts for the [API Docs APIs](https://api-docs.io/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [API Docs APIs](https://api-docs.io/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://api-docs.io/)- [Developer](https://api-docs.io/)- [Twitter](https://twitter.com/API_Docs)The [APIs.json](https://github.com/api-evangelist/api-docs/blob/master/apis.json) index provides links to all the relevant links for the API Docs APIs, including the OpenAPI and Postman if they are present.
